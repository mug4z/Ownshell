# Ownshell
A little shell wrote in C.

A little project, I did for learning C.

I will add some features.. maybe.

## How to run it.
1. Clone the repo 

```
git clone https://github.com/mug4z/Ownshell.git
```

2. Compile it

```
cd Ownshell
gcc -o main main.c
```

3. Run it
```
./main
```
